## Installing for termux

* `termux-setup-storage`
* `apt full-upgrade`
* `apt install git`
* `git clone https://github.com/mishakorzik/AllHackingTools`
* `cd AllHackingTools`
* `bash Install.sh`

## Installing for linux

* `apt upgrade`
* `apt install git`
* `git clone https://github.com/mishakorzik/AllHackingTools`
* `cd AllHackingTools`
* `bash Install.sh`

## Installing for ubuntu

* `apt-get upgrade`
* `apt-get install git`
* `git clone https://github.com/mishakorzik/AllHackingTools`
* `cd AllHackingTools`
* `bash Install.sh`
